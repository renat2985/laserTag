# laserTag
Laser Tag game on NodeMCU and SSD1306 display.

### If you want to see all the participants and change their weapons. You need to make an access point on your phone with the name "Point", and the password 12345678
### Further, all devices will appear on this site: http://www.onclick.lv/lasertag/

# Upload instruction
## Specification .bin files

    Module: Generic ESP8266 Module
    Flash Size: 1M
    CPU Frequency: 80Mhz
    Flash Mode: dio
    Flash Frequency: 40Mhz
    Upload Speed: 921600

### You can download the bin file here: [/build/esp8266.esp8266.generic/laserTag.ino.bin](https://github.com/renat2985/laserTag/raw/main/build/esp8266.esp8266.generic/laserTag.ino.bin)

### NodeMCU Flasher
https://github.com/nodemcu/nodemcu-flasher
Download Release: [Win32](https://github.com/nodemcu/nodemcu-flasher/blob/master/Win32/Release/ESP8266Flasher.exe) or [Win64](https://github.com/nodemcu/nodemcu-flasher/blob/master/Win64/Release/ESP8266Flasher.exe).
